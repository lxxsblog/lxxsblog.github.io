判断结构

判断结构分为if/switch case 机构
if分为标准if/if else/if else if
具体我会分开讲

标准的if结构

代码😅

if (判断条件)
运行内容;
这样只能运行if下方的一句内容

if (判断条件)
    {
        执行语句;
    }

花括号内的内容都运行

可以说标准的if非常简单，符合if的判断条件就执行语句，就是这个意思。
附上可有可无的流程图😂。

if-else结构。

这个和if的区别在于，如果if判断条件生效会运行一段程序，不生效再运行另一段程序

if (判断条件)
    {
        执行程序;
    }
    else
    {
        执行程序;
    }

基本结构很简单，和if的差不多。
附上比较有用的流程图😂。。。

if - else if结构

if (运行条件)
    {
        运行语句;
    }
    else if (运行条件1)
    {
        执行语句;
    }
    else if (运行条件2)
    {
        执行语句;
    }
    else
    {
        执行语句;
    }

哈哈，经典代码。
else if可以无限写的，所以这相当于很多if语句的嵌套，当这些条件都不符合时，执行else，非常好理解吧。

for循环语句

for ( int i = 0; 停止条件; i++ )
{
   你要运行的语句;
}

我是这么理解的，首先定义一个i的初值，然后定义一个i结束的值，循环条件成立并执行玩循环语句后i值的变化情况。
当然着些条件都可以不写，但是看你的需求了，如果你想无限循环那就不写。比如你就是想让他循环，再执行语句的部分添加break跳出也可以，当然最后我会补充break和continue。
正常for的判断次数比运行语句多一次，i的值和循环次数一样。
